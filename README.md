## About the Project

This repository contains the source code for the **Official Website and Interactive Curriculum Vitae (CV)** of developer **Ali Al-Karrar**.

The website is designed as a single-page application to showcase experience, skills, and projects in a modern and engaging manner. It features a dark theme with gold accents and includes advanced dynamic effects (such as particle effects and animated backgrounds) to provide a unique user experience.

## üöÄ Key Features

*   **Interactive CV:** An organized display of personal information, experience, and skills.
*   **Professional and Attractive Design:** Utilizes a dark theme with CSS animations and visual effects to capture attention.
*   **Arabic User Interface (RTL):** Full support for the Arabic language and Right-to-Left direction.
*   **Advanced Visual Effects:**
    *   Custom loading screen (Loader).
    *   Animated particle background.
    *   Hover effects on social media cards.
*   **Static Technology:** The website is built entirely with core web technologies (HTML, CSS, JS), making it extremely fast and easy to host.
